## ✨ Features

- 📝 Create multiple notes
- ✏️ Edit note titles and content
- 💾 Auto-save notes
- 📂 Sidebar navigation
- 🔍 Search notes instantly
- 🗑️ Delete notes
- ⚡ Lightweight and fast
- 💻 Responsive popup UI
- 🔒 Persistent storage using Chrome Storage API

---

## 💡 Why Mini Notebook?

Mini Notebook is designed for users who need a quick and efficient place to save ideas without leaving the browser.

Perfect for:

- 👨‍🎓 Students taking lecture notes
- 👨‍💻 Developers storing snippets and TODOs
- 💼 Professionals managing quick reminders
- ✍️ Writers capturing ideas instantly

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/mini-notebook.git
```

### 2. Open Chrome Extensions

```
chrome://extensions/
```

### 3. Enable **Developer Mode**

### 4. Click **Load unpacked**

### 5. Select the project folder

The extension is now ready to use.

---

## 🚀 Usage

1. Click the **Mini Notebook** extension icon.
2. Create a new note.
3. Write your content.
4. Notes are automatically saved.
5. Search or delete notes whenever needed.
